# Customer Segmentation Analysis

This project analyzes customer data to identify different customer segments based on purchasing behavior.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Machine Learning
- Jupyter Notebook

## Project Objectives
- Analyze customer spending behavior
- Identify customer segments
- Apply clustering techniques
- Visualize customer groups

## Techniques Used
- Data cleaning
- Exploratory Data Analysis (EDA)
- Customer segmentation
- Data visualization

## Project Files
- customer_segmentation_analysis.ipynb → full analysis
- customer_segmentation.html → exported notebook
- dataset.csv → dataset used in analysis

## Goal
Help businesses understand different customer groups to improve marketing strategies and decision-making.
